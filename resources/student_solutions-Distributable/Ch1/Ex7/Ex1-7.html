Running the command 

javadoc Greeter.java
creates the following files: 
Greeter.html
allclasses-frame.html
allclasses-noframe.html
constant-values.html
deprecated-list.html
help-doc.html
index-all.html
index.html
overview-tree.html
package-frame.html
package-list
package-summary.html
package-tree.html
stylesheet.css

Removing the documentation comments on the method and the constructor and then rerunning the javadoc command produces the same set of files, most of which are not affected by the change. The file Greeter.html still contains a listing for each method and constructor, however it no longer contains the method and parameter descriptions that are found in the comments that were removed.
