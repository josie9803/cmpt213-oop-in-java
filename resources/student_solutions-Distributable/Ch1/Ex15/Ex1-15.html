If the current directory is foo, then the java source files should be organized as follows: 

    foo/GreeterTest.java 

    foo/edu/sjsu/cs/CS200/johnDoe/Greeter.java
The following commands (executed from directory foo) will compile Greeter and GreeterTest and execute GreeterTest. 
    javac GreeterTest.java
    javac GreeterTest
After compilation, the files will be organized as follows: 
    foo/GreeterTest.java 
    foo/GreeterTest.class 

    foo/edu/sjsu/cs/CS200/johnDoe/Greeter.java 
    foo/edu/sjsu/cs/CS200/johnDoe/Greeter.class
