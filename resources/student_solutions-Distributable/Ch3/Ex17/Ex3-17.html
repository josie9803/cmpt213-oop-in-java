Immutable classes from the java standard library include the following:
  1. String (as mentioned in the text)
  2. java.io.File
  3. wrapper classes Integer, Double, etc.
  4. java.awt.Color
  5. java.awt.Font
  6. java.util.Currency
  7. java.util.Locale
  8. java.net.InetAddress
  9. java.net.URI (but not java.net.URL)
