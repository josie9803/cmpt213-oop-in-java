• Total orderings on String objects:
    1. The most commonly used total ordering is the lexicographic (ie dictionary) order of the string. This is the ordering used by the method compareTo().
    2. Another total ordering can be achieved as follows: Write the string s with characters s0s1. . .sn-1sn  as a number in base 65536, i.e. n(s) = Σ si 65536i. Then define s ≤ t iff n(s) ≤ n(t)

  • Partial orderings on String objects:
    1. Order by length: for Strings s and t, s ≤ t iff s.length() ≤ t.length().
