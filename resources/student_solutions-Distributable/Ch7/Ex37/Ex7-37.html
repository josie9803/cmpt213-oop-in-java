The steps carried out in the builder tool (NetBeans 4.1) are:
  1. If you have not already done so, choose the Tools -> Palette Manager menu option and install the Carbean.jar file. Place the bean into the Beans panel. 
  2. Select File -> New Project to create a new project of type "Java Application".
  3. Right-click on the project and select New -> JFrame form.
  4. In the “Swing” palette, click on the JSlider button. Then click on the bottom of the JFrame. A slider is now added to the form.
  5. Repeat the previous step to add another slider to the side of the form. Use the property sheet to change its orientation to vertical.
  6. In the “Beans” palette, click on the CarBean button.
  7. Click in the center of the JFrame. The CarBean is now added to the form.
  8. In the builder tool, right-click on the bottom slider and select the menu option Events → Change → stateChanged.
  9. The builder tool pops up the source window and positions your cursor next to a comment
    // TODO Add your handling code here:
  10. Add the code
    carBean1.setX(jSlider1.getValue());
  11. Click on the Design button to return to the form designer.
  12. Repeat the preceding steps for the other slider, this time adding the code
    carBean1.setY(jSlider2.getValue());
  13. Set the value properties of both sliders to 0.
  14. Select Run -> Run File to execute the program. Now the sliders move the car position.
